# Anvil Level Design

## Intro

Anvil Level Design (Anvil LD or Anvil) is an addon for blender that is intended to speed up and support video game level design. It is particularly inspired by Trenchbroom and Source 1 and 2 Hammer. 

Anvil combines material application and adjustment tools, navigation UX (including 3d view navigation and specialised material and prefab browsers), geometry helpers including blockout tools, and a slew of miscellaneous UX improvements.

Anvil is currently engine agnostic; levels are intended to be exported to .fbx and .gltf / .glb. Anvil currently includes .glb export helper functions. In future Anvil will include .fbx helper functions and may include reference implementations for game engine import workflows.

LLM coding assistance is used in the development of Anvil.
